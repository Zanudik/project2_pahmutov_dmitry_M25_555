# Primitive DB

**Primitive DB** — это простая текстовая база данных на Python, которая позволяет создавать таблицы, добавлять, изменять, удалять записи и выполнять запросы через консольный интерфейс.

## Описание проекта

- Проект работает в терминале и не требует внешних СУБД.
- Поддерживаются типы данных: `int`, `str`, `bool`.
- Доступные операции:
  - Создание и удаление таблиц.
  - Добавление, обновление и удаление записей.
  - Выборка записей с фильтром `where`.
  - Просмотр информации о таблице.
- Данные сохраняются в формате JSON в папке `data`.

## Установка

```bash
make install
```

## Запуск проекта
```bash
make project
```

## Примеры команд
```text
Введите команду: create_table users name:str age:int is_admin:bool

Таблица "users" успешно создана со столбцами: ID:int, name:str, age:int, is_admin:bool

Введите команду: insert into users values ("Alice", 30, false)

Функция insert выполнилась за 0.001 секунд.
Запись успешно добавлена в таблицу "users".

Введите команду: select from users

Функция select выполнилась за 0.000 секунд.
+----+-------+-----+----------+
| ID |  name | age | is_admin |
+----+-------+-----+----------+
| 1  | Alice |  30 |  False   |
+----+-------+-----+----------+

Введите команду: update users set age = 31 where name = "Alice"

Обновление выполнено.

Введите команду: delete from users where name = "Alice"

Вы уверены, что хотите выполнить "удаление записи"? [y/n]: 

y

Удаление выполнено.

Введите команду: drop_table users

Вы уверены, что хотите выполнить "удаление таблицы"? [y/n]
: 

y

Таблица "users" успешно удалена.

Введите команду: exit

Выход.
```
